# Sistemas
Sistema de criação de contas, onde o usuário é capaz de consultar sua conta, depositar e sacar

# Funcionalidades

- Criar conta
- Depositar dinheiro
- Sacar dinheiro
- Consultar saldo

# Conceitos utilizados

- Funções
- Dicionários
- Estruturas condicionais
- Laços de repetição
- Entrada e saída de dados
