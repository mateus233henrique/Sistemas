# Caso de Teste 1– Criar conta (Funcionalidade)
objetivo: Validar a funcionalidade “criar conta” no sistema 

Pré-condição: Acesso ao sistema

Passos:
    1.	Acessar o pycharm 
    2.	Codar o sistema
    3.	Informar o nome do titular

Resultado esperado: Exibir mensagem “Conta criada com sucesso.”

# Caso de Teste 2– Depositar (Funcionalidade)
Objetivo: Validar se a opção “depositar” identifica o nome da conta ao ser digitado e inserir o valor depositado pelo titular

Pré-condição: titular deve está com a conta criada

Passos:
    1.	Acessar o pycharm 
    2.	Codar o sistema
    3.	Informar o nome do titular cadastrado
    4.	Informar o valor desejado para ser depositado na conta cadastrada

Resultado esperado: Exibir mensagem “Depósito realizado com sucesso.”

## Caso de Teste 4– Consultar saldo (Funcionalidade)
Objetivo: Validar se a opção “consultar saldo” identifica o nome da conta ao ser digitado pelo titular pode ser consultado e verificar se está de acordo com os valores depositados e sacados no saldo atual

Pré-condição: titular deve está com a conta criada, com valores depositados e sacados

Passos:
    1.	Acessar o pycharm 
    2.	Codar o sistema
    3.	Informar o nome do titular cadastrado

Resultado esperado: exibir mensagem “Saldo atual: R$ x.x”.
